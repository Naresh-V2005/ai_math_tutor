# 📘 AI Math Tutor

AI Math Tutor is an intelligent web application that can **analyze, interpret, and solve mathematical problems from images** using Optical Character Recognition (OCR) and Artificial Intelligence.

It allows users to upload handwritten or printed math problems and receive **accurate, step-by-step solutions** instantly.

---

## 🌟 Overview

Solving math problems manually can be time-consuming, especially when dealing with handwritten equations. AI Math Tutor simplifies this process by combining:

- 📷 Image Processing
- 🔍 OCR (Optical Character Recognition)
- 🧠 AI-based reasoning
- 🧮 Symbolic computation

This project demonstrates the integration of **Computer Vision + AI + Web Development** into a real-world application.

---

## 🚀 Key Features

### 📷 Image-Based Input
- Upload handwritten or printed math problems
- Supports common image formats (JPG, PNG)

### 🔍 OCR Extraction
- Extracts mathematical expressions using:
  - Tesseract OCR
  - EasyOCR
- Handles noisy and handwritten inputs

### 🧠 AI-Powered Solver
- Uses AI models (Gemini / OpenAI) to:
  - Understand equations
  - Generate step-by-step solutions
  - Provide explanations

### 🧮 Symbolic Computation
- Uses SymPy for:
  - Equation solving
  - Simplification
  - Algebraic manipulation

### 📄 LaTeX Conversion
- Converts extracted math into LaTeX format
- Improves readability and formatting

### ⚡ Interactive UI
- Simple drag-and-drop image upload
- Real-time preview and results display
